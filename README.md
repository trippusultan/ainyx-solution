# Ainyx Solution — Product UI/UX Intern Task

Figma recreation for the Product UI/UX Intern assignment.

## Pages
- Canvas Task — task list, filters, detail panel
- Form Task — form fields, validation structure, review screen

## Setup

```bash
npm install
npm run dev
```

## Notes
- Built with React + Vite
- Component-based structure
- One component per reusable Figma element (buttons, inputs, cards, badges)
- Auto Layout emulation with flexible React layouts

## Deliverables
- `/canvas-task` — Canvas Task page
- `/form-task` — Form Task page
